# Videos required here

Place two videos in this folder (both referenced by exact filename in the components):

1. `hero.mp4` — the 10-second scroll-scrubbed video used by `src/components/Hero/Hero.jsx`.
2. `craft.mp4` — the second scroll-controlled video used by `src/components/ScrollVideo/ScrollVideo.jsx`.

Recommended format: H.264 MP4, 1920x1080 or larger, no audio needed (videos are muted
and scrubbed by scroll position, never auto-played). Keep file size reasonable
(under ~15MB) since the whole file loads before scrubbing starts smoothly.
